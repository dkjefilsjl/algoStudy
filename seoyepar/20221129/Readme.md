<문제 요약>

- 길이가 같은 두개의 큐.
- 하나의 큐를 골라 원소 추출.
- 추출된 원소를 다른 큐에 집어넣음.
- 각 큐의 원소의 합이 같도록 만듦.
- return : 이때 필요한 작업의 최소 횟수는?
- pop과 insert를 합쳐서 1회로 간주.
  - pop을 하면 배열의 첫번째 원소가 추출.
  - insert를 하면 배열의 끝에 원소가 추가.
